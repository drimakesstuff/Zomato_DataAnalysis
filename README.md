## **Usage**

1. **Clone the repository**:
```bash
git clone https://github.com/drimakesstuff/Zomato_DataAnalysis.git
```
2. Install dependencies:
```
bash

pip install pandas matplotlib jupyterlab
```
3. Run the notebook:
```
bash
jupyter lab
```
4. Open zomato.ipynb and run the cells.
